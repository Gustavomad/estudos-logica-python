# Resumo: Aula 01 - Lógica de Programação

- **O que aprendi:** A diferença entre algoritmo e linguagem de programação.
- **Definição de Algoritmo:** É a sequência lógica de passos para resolver um problema.
- **Definição de Código:** É a implementação do algoritmo em uma linguagem específica (como Python) para que o computador execute a tarefa.
